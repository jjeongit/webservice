# spring-boot + JPA
OAuth2.0를 활용하여 구글,네이버 로그인 구현하기

## 프로젝트 소개
요즘 회원가입보다 간편로그인을 자주 하는 니즈에 맞춰서
OAuth 2.0을 활용하여 구글, 네이버 로그인 구현해보았습니다.
<br>

## 개발기간
1인 프로젝트, 2week

### 개발 환경
'JAVA 17'

'JDK 1.8'

'gradle 4.10'

**IDE** : IntelliJ IDEA

**Framework** : stringboot 2.1.7

**Database** : H2 database

**ORM** : JPA

## 주요기능
처음 사용해보는 JPA를 사용해 간단하게 데이터 처리를 하여 OAuth 2.0을 활용한 로그인 기능 구현
